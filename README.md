# PDF Editor Pro

**Acrobat Pro 互換のオフライン Web PDF エディター**

すべての処理がブラウザ内で完結します。ファイルが外部サーバーへ送信されることは一切なく、初回読み込み後は完全オフラインで動作します(Service Worker によるキャッシュ + 全ライブラリ同梱)。

## 起動方法

```bash
node server.js        # → http://localhost:8080 を開く
# または
npm start
```

Node.js があれば依存インストール不要で起動できます(`server.js` は依存ゼロ)。初回アクセス時に Service Worker が全アセットをキャッシュするため、2回目以降はネットワークなしでも動作します。

## 使い方の要点

- **開く**: ドラッグ&ドロップ、Ctrl+O、またはようこそ画面のボタン。PDF以外(Office/画像)を開くと自動でPDFへ変換されます。
- **編集**: 右パネル「PDFを編集」→「テキストと画像を編集」。ページ上のテキストをクリックして書き換え、Enterで確定。画像はクリックして「差し替え/削除」。
- **注釈**: ツールバーの注釈ツールで描画 → 「注釈を適用」でPDFへ書き込み。未適用の注釈はダブルクリックで削除できます。
- **ページ整理**: 右パネル「ページを整理」。グリッド上でドラッグ並べ替え、Ctrl+クリックで複数選択。
- **保存**: 保存はブラウザのダウンロードとして行われます(Webアプリのため上書きはダウンロード先で置換)。

## アーキテクチャ

```
index.html          UIシェル(メニュー/ツールバー/パネル/ダイアログ)
css/app.css         Acrobat Pro DC風ダークテーマ
js/
  state.js          単一の信頼できる情報源 = PDFバイト列。undo/redo、pdf.js再読込
  viewer.js         pdf.js連続スクロール表示・サムネイル・検索・印刷
  organize.js       ページ整理(pdf-lib)・結合・分割
  annotate.js       注釈オーバーレイ → pdf-libでPDFへ書き込み
  edit.js           テキスト/画像編集(オペレーターリストで画像位置検出、白塗り+再描画)
  convert.js        作成(mammoth/SheetJS/JSZip/foreignObjectレンダリング)・書き出し(docx/xlsx/pptxgen)
  ocr.js            Tesseract.js(日英)→不可視テキスト埋め込み
  compare.js        pixelmatchピクセル差分+LCSテキスト差分
  protect.js        暗号化(@cantoo/pdf-lib)・圧縮・プロパティ
sw.js               オフラインキャッシュ
server.js           依存ゼロのローカルサーバー
vendor/             全ライブラリ・日本語フォント・OCR辞書(オフライン用に同梱)
```

**編集方式について**: 既存要素の編集は「白塗り+再描画」方式です(元のコンテンツストリームを直接書き換えるのではなく、上から白い矩形を重ねて新しい内容を描画)。これは多くの軽量PDF編集ツールと同じアプローチで、背景が白以外のページでは編集跡が見える場合があります。

## 既知の制限

- PowerPoint→PDF はテキスト抽出ベースの簡易変換(レイアウト・図形は再現されません)
- PDF→Word/Excel はテキスト抽出ベース(段組・表構造の完全再現はしません)
- 画像圧縮モード(150/110/72dpi)はページを画像化するためテキスト情報が失われます(「軽量化のみ」はテキスト保持)
- 注釈は「適用」後はページコンテンツに書き込まれるため、Acrobatの注釈オブジェクトとして再編集できるのはノート注釈(💬)のみ
- フォームフィールドの作成・電子署名は未対応(今後の候補)
